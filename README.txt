This README file explains what each file does and how to run them.

1. motivation.ipynb

This file shows results on why we were motivated to take this project and work.

2. data_processing.ipynb

This file explores some exploratory data analysis on the data and shows some result.

3. without_feature_engg.ipynb

This file shows model results without feature engineering.

4. with_feature_engg.ipynb

This file shows model results with feature engineering.


HOW TO RUN THEM
===============

Open command prompt from this current folder.

1. Type jupyter notebook

2. Wait till the localhost:8888 opens.

3. Click the file that you would like to open and run.

4. Run each cell sequentially to get results.
